Rutas de NeoMapas

Las transectas o rutas de NeoMapas representan unidades regionales de análisis (URA), dentro de las cuales se distribuyen las unidades elementales de muestreo (UEM) que varían según el grupo taxonómico (trampas, punto observación, recorrido cronometrado).
